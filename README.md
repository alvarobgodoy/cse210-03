# Jumper
Jumper is a game in which the player seeks to solve a puzzle by guessing the letters of a secret word one at a time.

## Project Structure
```
root                    (root folder)
+-- seeker              (source code)
  +-- game              (classes)
  +-- __main__.py       (start program)
+-- README.md           (general info)
```
